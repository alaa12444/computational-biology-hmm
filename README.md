# Computational Biology Assignment

## Project Overview
This project performs Multiple Sequence Alignment (MSA) using MAFFT and constructs a Profile Hidden Markov Model (HMM).

## Steps
1. FASTA sequence preparation
2. MAFFT alignment
3. Gap fraction analysis
4. Profile HMM construction

## Tools
- MAFFT
- Python (Biopython)

## Output
- Aligned sequences
- Gap classification table
- HMM state paths
- Final PDF report
